# PrivacyPolicy
 D.O.N.T.P.A.N.I.C uses localStorage solely for storing the user's dark mode preference locally within their browser. This data is
     not transmitted or shared.
   * The extension accesses the user's bookmarks via the chrome.bookmarks API, but this data remains local to the user's browser
     and is not collected, processed, or transmitted externally.
   * background.js only handles opening the side panel and does not engage in any data collection.
